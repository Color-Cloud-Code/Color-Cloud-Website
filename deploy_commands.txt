# Color Cloud: Einfache Anleitung zum Veröffentlichen

Deine Webseite ist direkt mit Cloudflare Pages verbunden. Du musst nur Git-Befehle nutzen.

## 1. Live-Seite aktualisieren (Production)
Alles, was du hierhin schickst, ist sofort für jeden unter deiner Haupt-URL sichtbar.

$ git add .
$ git commit -m "Webseite aktualisiert"
$ git push origin main

---

## 2. Vorschau-Seite erstellen (Preview)
Wenn du etwas testen willst, ohne die Live-Seite zu ändern, schickst du es an einen "Test-Zweig" (Branch). Cloudflare gibt dir dann automatisch einen eigenen Link nur für diese Vorschau.

# Einmalig einen Test-Zweig erstellen und hochladen:
$ git checkout -b vorschau
$ git add .
$ git commit -m "Test der neuen Funktionen"
$ git push origin vorschau

# Wenn du später wieder etwas in die Vorschau schicken willst:
$ git add .
$ git commit -m "Weitere Tests"
$ git push origin vorschau

# So kommst du zurück zur Haupt-Seite:
$ git checkout main

---

## Zusammenfassung
- Schickst du an 'main' -> Live-Seite (Live für alle)
- Schickst du an 'vorschau' -> Test-Seite (Nur über speziellen Link von Cloudflare)
